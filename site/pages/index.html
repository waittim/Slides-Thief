Slides Thief — Convert Angled Slide Photos to PDF

Detect slide boundaries, correct perspective distortion, and create one PDF locally in your browser without uploading source photos.

Supported images

JPEG, PNG, WebP, HEIC, and HEIF.

Automatic and manual correction

Detect four corners automatically, then drag any handle that needs adjustment.

CLI and machine-readable reports

The Python CLI supports batch processing, manual correction files, and JSON detection reports.
